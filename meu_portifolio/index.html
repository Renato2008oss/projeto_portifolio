Jogo Seven Eleven: Um algoritmo básico de um jogo de dados

Este projeto apresenta um algoritmo simples de um jogo de dados chamado "Seven Eleven". O objetivo é simular a jogabilidade básica utilizando lógica de programação e elementos aleatórios.

No jogo, dois dados são lançados, e o jogador vence automaticamente se a soma dos dados for 7 ou 11 na primeira jogada. Caso contrário, o jogo segue para uma segunda fase onde regras adicionais determinam vitória ou derrota com base em novos lançamentos.

O algoritmo foi implementado em python e é um esboço da belaza da programção

O projeto serve como introdução à programação de jogos simples e ao uso de elementos aleatórios em lógica condicional, proporcionando aprendizado prático em desenvolvimento web interativo.

Algoritmo de pesquisa binária

Este projeto demonstra o funcionamento do algoritmo de pesquisa binária, uma técnica eficiente para encontrar elementos em listas ordenadas. A pesquisa binária reduz progressivamente o intervalo de busca, descartando metade dos elementos a cada passo.

O algoritmo funciona comparando o elemento desejado (chave de busca) com o valor central da lista. Se forem iguais, a busca termina. Se a chave for menor, a busca continua na metade inferior; se for maior, na metade superior. O processo se repete até o elemento ser encontrado ou a lista se esgotar.

Para que a pesquisa binária funcione corretamente, é essencial que a lista esteja previamente ordenada. O tempo de execução é O(log n), tornando-a muito mais rápida que a busca linear (O(n)) em listas grandes.

O projeto foi implementado com NodeJS e serve apenas como um visualizador de como a pesquisa binária funciona

Roleta Russa: Um algoritmo feito em C

Este jogo é uma simulação educativa que alerta sobre os riscos de executar códigos desconhecidos ou maliciosos. Apesar de parecer simples, ele representa o que pode acontecer se um script mal-intencionado for executado no seu sistema. A seguir, estão os possíveis efeitos simulados no jogo:

  • 1 - Remoção do diretório System32: Essa pasta é essencial para o funcionamento do Windows. Removê-la pode tornar o sistema inutilizável, exigindo uma reinstalação completa.
  • 2 - Envio de malware que altera o idioma do sistema: O jogo pode simular o envio de um programa que modifica o idioma do sistema operacional, dificultando a navegação e compreensão do ambiente para o usuário.
  • 3 - Desligamento forçado da máquina: O script simula a execução de um comando que desliga o computador sem o seu consentimento, o que pode resultar em perda de dados não salvos.
  • 4 - Injeção de malware espião: Representa um cenário onde um programa é instalado silenciosamente para coletar informações pessoais, como arquivos, credenciais salvas e senhas de Wi-Fi.
  • 5 - Abertura de múltiplas janelas do sistema: Essa ação simula uma sobrecarga no sistema, abrindo várias abas ou aplicativos de forma descontrolada, o que pode travar a máquina.
  • 6 - Execução de um laço infinito: Um loop sem fim pode consumir recursos como memória RAM e CPU, tornando o sistema lento ou até inutilizável até que seja forçadamente desligado.

⚠️ Importante: Este projeto é apenas para fins didáticos. Nenhuma dessas ações é realmente executada. O objetivo é ensinar, de forma lúdica, os perigos de interagir com softwares de origem duvidosa e a importância de verificar o código antes de executá-lo.
